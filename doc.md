##汇编器

###使用方法

    python3 encoder.py input [output]

###算法简介

- 将汇编文件分行读入
- 替换每行的非必要字符
- 将每条指令分词存储
- 使用正则表达式提取必要信息
- 与存储的关键字进行匹配并转换为汇编码
- 将汇编码转换为二进制输出到文件中

##反汇编器

###使用方法

    python3 decoder.py input [output]

###算法简介

- 将二进制文件读入
- 将每32bit转换为一个
- 将32bit进行处理并转换为汇编指令
- format汇编指令
- 将汇编指令输出到文件中
